### Seznami v Pythonu so **spremenljivi**

.source[```python
seznam = [1, 2, 3]
def f(x):
    seznam.append(x)
    return len(seznam)
def g(x):
    return f(x) + f(x)
```]

.terminal[```python
>>> g(3)
9
>>> f(3) + f(3)
13
>>> g(3)
17
```]

### Reference v OCamlu so **spremenljive**

.source[```
